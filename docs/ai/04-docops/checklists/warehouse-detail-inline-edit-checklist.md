# Warehouse Detail Inline Edit Checklist

- [ ] Detail page remains on `/warehouses/{id}` when Edit is clicked.
- [ ] Edit form opens as a slideover/modal.
- [ ] Form loads the selected Warehouse by explicit `recordId`.
- [ ] Save updates the record.
- [ ] After save, detail page refreshes and displays updated data.
- [ ] Closing the modal does not navigate to index.
